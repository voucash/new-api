# New API 支付设置

`QuantumNous/new-api` 接入 VouPay 时，只需要在后台支付设置页填写以下内容。

![支付设置](./settings.png)

## 必填项

| 配置项 | 填写内容 |
| --- | --- |
| 支付地址 | `https://voupay.io/epay` |
| 易支付商户 ID | 在 <https://voucash.com/dev/dashboard> 获取 |
| 易支付商户密钥 | 在 <https://voucash.com/dev/dashboard> 获取 |

注册后可在“我的应用”中查看商户ID和密钥。
